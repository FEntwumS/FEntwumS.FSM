# Step 1
- Download and Install the following:
	- Visual Studio Code(https://code.visualstudio.com/)
	- OneWare Studio (https://one-ware.com/de/docs/getting-started/setup)
	- JDK (e.g. any current JDK https://adoptium.net/)
		- in the Installation Wizard enable setting JAVA_HOME PATH for all users

# Step 2
1. Open Visual Studio Code
2. Search bar: ">.net install "
3. Wait for suggestion e.g. version 10.0.300
4. Press enter and proceed with the sdk installation
5. Restart Visual Studio Code after finishing the .NET installation
6. Clone the repository with 
	"git clone https://github.com/FEntwumS/FEntwumS.FSM.git"
7. Open the cloned folder "FEntwumS.FSM" in Visual Studio Code 
8. Status bar on the bottom should be "non-red" and no error message visible
# Step 3
1. Navigate to Terminal to click on "Build task"
2. Press the "Run and Debug" button on the left sidebar(4th entry)
3. Click on the red play button next to run plugin
4. This should open OneWare Studio with the Plugin installed
